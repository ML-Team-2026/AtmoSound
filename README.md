## How to Run the the AtmoSound Streamlit App

1. Clone the repository:
   git clone [https://github.com/<your-repo-link>](https://github.com/ML-Team-2026/AtmoSound.git)

2. Navigate to the project folder:
   cd AtmoSound

3. Make sure the following files are in the same folder:
   - app.py
   - Gym pic.webp  
   (The image is required for the homepage to display correctly)

4. Install Streamlit:
   pip install streamlit

5. Run the app:
   streamlit run app.py

6. Open the local URL shown in the terminal (usually http://localhost:8501) in your browser.

7. In the app:
   - Enter the Cornell tech Cafe google map link: "https://www.google.com/maps/search/Cornell+Tech+Cafe" 
   - It will take you to the statistics page link which is interactive
   - Click "Go To Playlist" to see songs generated for Cornell tech Cafe
   - To return home click either 'Home' button or AtmoSound logo
